# App Builder - Orquestador de Creación de Aplicaciones

Esta skill actúa como un **Project Manager técnico** para construir aplicaciones completas. Analiza tus requisitos, define el stack tecnológico y coordina a otros agentes (Frontend, Backend, Database) para ejecutar el proyecto.

## 🚀 Funcionalidades Principales

*   **Detección de Proyectos**: Identifica automáticamente si necesitas un E-commerce, SaaS, Blog, etc.
*   **Selección de Tecnología**: Elige el mejor stack para 2025 (Next.js, Supabase, Tailwind, etc.) o se adapta a tus preferencias.
*   **Planificación Estructurada**: Crea un plan detallado (API, BD, UI) antes de empezar a escribir código.
*   **Coordinación de Agentes**: Delega tareas específicas a skills especializadas.
*   **Plantillas (Scaffolding)**: Genera la estructura inicial optimizada para múltiples tipos de apps (Web, Móvil, Desktop, CLI).

## 📋 ¿Cuándo usar esta skill?

Úsala al **inicio de cualquier proyecto nuevo** o cuando necesites agregar **grandes funcionalidades** que requieran planificación arquitectónica.

Ejemplo: *"Quiero crear un clon de Airbnb con reservas y pagos"* -> **App Builder** se encargará de diseñar la base de datos, la API y dividir el trabajo.
