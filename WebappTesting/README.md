# Pruebas de Aplicaciones Web (Webapp Testing)

Kit de herramientas para automatizar e interactuar con aplicaciones web locales usando Playwright, permitiendo verificar funcionalidad y depurar la UI.

## 🚀 Cuándo utilizarla
- Después de hacer cambios visuales o funcionales en el frontend.
- Para verificar que los flujos críticos (login, formularios, carritos) no se hayan roto.
- Cuando necesites que el agente "vea" lo que está pasando en la web para encontrar un bug de CSS.

## 💡 Ejemplos de uso
- "Verifica que el botón de 'Enviar' en localhost:3000 sea clicable y envíe los datos."
- "Comprueba si la barra de navegación es responsive en monitores pequeños."
- "Ejecuta una prueba para ver si el mensaje de error aparece cuando el login falla."
