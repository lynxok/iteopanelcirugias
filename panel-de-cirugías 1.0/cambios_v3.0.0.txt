
RESUMEN DE CAMBIOS - VERSIÓN 3.0.0
Fecha: 07/05/2026

HITOS PRINCIPALES:
1. ARQUITECTURA HÍBRIDA: El sistema ahora detecta si se ejecuta como Web o como App de Escritorio (Electron).
2. AUTOMATIZACIÓN OSER: Integración total con el portal de OSER para extraer fechas de autorización y prácticas.
3. AUDITORÍA INTELIGENTE: Nuevo modal que compara los datos de OSER con los de la App, permitiendo revisiones manuales antes de aplicar cambios.
4. VALIDACIÓN DE PRÁCTICAS: Lógica de comparación por palabras clave para detectar discrepancias entre el procedimiento agendado y el autorizado.
5. PUENTE DE COMUNICACIÓN (BRIDGE): Implementación de un servidor local (Node.js) para permitir que la versión Web local ejecute scripts de Python.

MEJORAS DE UI:
- Nueva pestaña "Sincronización OSER" en Configuración.
- Banners inteligentes de "Descarga de App Desktop" para usuarios web.
- Modal detallado de diferencias con estados visuales (✅/❌).
