# Panel de Cirugías ITEO - Distribución

Este repositorio se utiliza exclusivamente para la distribución de actualizaciones del instalador de escritorio (`.exe`).

## Actualizaciones
Las actualizaciones se gestionan a través de la sección de **Releases**.
El sistema `electron-updater` de la aplicación consulta este repositorio para detectar nuevas versiones.
