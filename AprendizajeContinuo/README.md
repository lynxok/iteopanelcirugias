# Aprendizaje Continuo

Gestiona una base de conocimiento persistente para el proyecto, registrando decisiones técnicas y patrones para mantener el contexto entre sesiones.

## 🚀 Cuándo utilizarla
- Al finalizar una tarea importante para guardar las decisiones tomadas.
- Cuando cambies de arquitectura o elijas una nueva herramienta (ej: cambiar de Prisma a Drizzle).
- Para evitar repetir explicaciones sobre cómo funciona una parte del código en el futuro.

## 💡 Ejemplos de uso
- "Registra en nuestra base de conocimiento que ahora usaremos UUIDs para todas las claves primarias."
- "Actualiza el KNOWLEDGE.md con el resumen de cómo configuramos el servidor de autenticación."
- "Recuerda que para este proyecto preferimos usar CSS nativo en lugar de Tailwind."
