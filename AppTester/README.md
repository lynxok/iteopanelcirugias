# AppTester - Senior QA Automation Engineer

Esta skill simula a un **QA Tester Senior** que analiza tu código no solo para encontrar bugs, sino para auditar la calidad integral del producto: Seguridad, Accesibilidad, Rendimiento y Mantenibilidad.

## 🚀 Funcionalidades de Nivel Senior

### 1. Auditoría de Calidad 360°
*   **Seguridad**: Detección proactiva de riesgos (XSS, Inyecciones, exposición de datos).
*   **Accesibilidad (A11y)**: Verificación de cumplimiento WCAG (contrastes, lectores de pantalla, teclado).
*   **Rendimiento**: Identificación de renders innecesarios, cuellos de botella y *layout shifts*.

### 2. Automatización Robusta
*   Generación de scripts (Playwright/Cypress) usando **Mejores Prácticas**.
*   Uso de selectores resilientes (evita que los tests se rompan por cambios de CSS).
*   Implementación de patrones como **Page Object Model**.

### 3. Estrategias de Prueba ("Edge Cases")
*   No se limita al "Happy Path". Diseña escenarios para romper la aplicación:
    *   Fallos de red / API.
    *   Inputs maliciosos.
    *   Concurrencia (doble click en botones de pago).

### 4. Benchmarking Competitivo
*   Analiza si a tu app le faltan funcionalidades estándar en el mercado.
*   *Ejemplo*: "Veo que tienes un Login, pero no veo opción de 'Olvidé mi contraseña' ni Login Social. Las apps modernas como Spotify o Airbnb ofrecen esto para reducir fricción."

## 📋 Ejemplos de Interacción

*   *"Analiza este componente `PaymentForm.tsx` y dime qué riesgos de seguridad y usabilidad ves."*
*   *"Genera una suite de pruebas E2E en Playwright para el flujo de Checkout, cubriendo casos de error de tarjeta y timeouts."*
*   *"Audita la accesibilidad de mi Landing Page y dime qué debo corregir para cumplir WCAG AA."*
