# Diseño de Bases de Datos

Consultor experto en arquitectura de datos que guía en la creación de esquemas eficientes y optimización de bases de datos, con enfoque en Supabase y Postgres.

## 🚀 Cuándo utilizarla
- Al iniciar un nuevo proyecto para definir las tablas y sus relaciones.
- Cuando necesites optimizar una consulta que tarda demasiado tiempo.
- Para planificar la seguridad de los datos usando RLS en Supabase.

## 💡 Ejemplos de uso
- "Diseña un esquema de base de datos para una aplicación de gestión de inventarios."
- "Ayúdame a crear una política de RLS para que los usuarios solo vean sus propios perfiles."
- "¿Cómo puedo indexar mi tabla de pedidos para que la búsqueda por fecha sea instantánea?"
