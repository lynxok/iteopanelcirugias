# UX/UI Design Auditor - Skill de Auditoría Senior

Esta skill transforma a tu asistente en un **Diseñador UX/UI Senior** con 10 años de experiencia. Su objetivo es analizar tus interfaces (ya sea código, descripciones o capturas) y proponer mejoras que eleven la calidad visual y la facilidad de uso al nivel de las mejores aplicaciones del mercado.

## 🚀 ¿Qué puede hacer por ti?

*   **Auditoría Estética**: Crítica profunda sobre colores, tipografía, espaciado y jerarquía.
*   **Análisis UX**: Optimización de flujos, reducción de carga cognitiva y aplicación de heurísticas.
*   **Tendencias 2025-2026**: Sugerencias para adelantarte al mercado con Generative UI, Spatial Design y neuro-inclusión.
*   **Check de Accesibilidad**: Asegura que tu app sea usable para todos.
*   **Prototipado Visual**: Genera mockups realistas antes de tocar una sola línea de código.

## 🛡️ Flujo de Trabajo Seguro

Para evitar cambios no deseados, la skill seguirá siempre este orden:
1. **Análisis Profundo** -> 2. **Mockup Visual** -> 3. **Tu Aprobación** -> 4. **Implementación en Código**.

## 📋 ¿Cómo usarla?

Simplemente pídele que analice una pantalla o componente:
*"Actúa como el UX Auditor y analiza este componente de Login. Busca cómo modernizarlo y mejorar la UX."*

## 🍱 Recursos Incluidos
- **Heurísticas Modernas**: Basadas en estándares de la industria.
- **Trend Radar 2024-2025**: Para que tu diseño nunca se vea anticuado.
- **WCAG Checklist**: Guía de accesibilidad integrada.
